# The Growth Breakfast — attendee directory

Static GitHub Pages site (HTML/CSS/JS) backed by Firebase (Cloud Firestore +
anonymous auth). See [README.md](README.md) for architecture, setup, and the admin flow.

## Commit & PR conventions (applies to ALL agents)

Do **not** add AI/agent attribution to commits, pull requests, or code. Specifically:

- **No** `Co-Authored-By:` trailers for AI agents (Claude, Cursor, Copilot, etc.).
- **No** "Generated with [Claude Code]" / "🤖 Generated with …" or similar generated-by lines.
- **No** agent names, links, or "written by AI" notes in commit messages, PR titles/bodies,
  or source comments.

Write commit messages as a normal human contributor would: describe **what** changed and
**why**, nothing more. This rule applies to every agent working in this repo
(Claude Code, Cursor, etc.).
