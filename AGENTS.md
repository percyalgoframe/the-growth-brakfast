# Agent rules — The Growth Breakfast

Guidance for any AI coding agent (Claude Code, Cursor, Copilot, etc.) working in this repo.
Project overview and setup live in [README.md](README.md).

## Commit & PR conventions

Do **not** add AI/agent attribution to commits, pull requests, or code:

- **No** `Co-Authored-By:` trailers for AI agents (Claude, Cursor, Copilot, etc.).
- **No** "Generated with [Claude Code]" / "🤖 Generated with …" or similar generated-by lines.
- **No** agent names, links, or "written by AI" notes in commit messages, PR titles/bodies,
  or source comments.

Write commit messages as a normal human contributor would — describe what changed and why.
